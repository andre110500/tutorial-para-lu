💻 Código completo del proyecto

Elegí un archivo, copialo o descargalo. Todo listo para usar.

Se descargarán los 5 archivos de código. Las imágenes están en codigo-completo/imagenes.

index.html proyectos.html contacto.html style.css README.md

📄 index.html — Página de inicio

📄 proyectos.html — Página de proyectos con tabla

📄 contacto.html — Página de contacto con formulario

📄 css/style.css — Todos los estilos CSS

📄 README.md — Documentación del proyecto

📁 Cómo armar la carpeta del proyecto

Después de descargar los archivos, creá esta estructura en tu computadora:

tu-proyecto/
├── index.html
├── proyectos.html
├── contacto.html
├── css/
│   └── style.css
├── imagenes/
│   ├── studio-lu-nails.png
│   ├── diseno-floral.png
│   ├── francesa-glitter.png
│   └── marmol-oro.png
└── README.md

Abrí cada archivo con el bloc de notas, pegá el código que copiaste y copiá también la carpeta de imágenes. ¡Listo!
