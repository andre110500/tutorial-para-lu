📖 Paso 1: ¿Qué es HTML y CSS?

Antes de escribir código, entendé de qué se trata.

🌐 ¿Qué es una página web?

Una página web es como un documento que vive en internet. Cuando la abrís en tu navegador (Chrome, Edge, etc.), ves texto, imágenes, colores y botones.

Esa página se construye con dos lenguajes principales:

📄 HTML (el esqueleto)

HTML es el contenido de la página: los títulos, párrafos, imágenes, tablas y formularios. Es como los huesos de un cuerpo.

Ejemplo de HTML: 
<h1>Hola, soy Lu</h1>
<p>Este es mi primer proyecto web.</p>

El navegador lo muestra como un título grande y un párrafo.

🎨 CSS (la ropa y los colores)

CSS es el diseño: colores, tamaños, fuentes y cómo se ordena todo en la pantalla. Es como la ropa, los colores y los accesorios del cuerpo.

Ejemplo de CSS: 
h1 {
    color: blue;
    font-size: 32px;
}

Esto hace que todos los títulos h1 sean azules y grandes.

📁 Estructura del proyecto

Tu proyecto va a tener esta carpeta con estos archivos:

tu-proyecto/
├── index.html        → Página de inicio
├── proyectos.html    → Página de proyectos (con tabla)
├── contacto.html     → Página de contacto (con formulario)
├── css/
│   └── style.css     → Archivo de estilos
└── imagenes/
    └── (tus fotos)   → Tus imágenes

Todo esto se sube a GitHub para entregarlo.

🔑 Regla de oro

Todo lo que está entre <etiqueta> y </etiqueta> es HTML. Todo lo que está fuera es CSS. ¡No te compliques!

💡 Recordá: HTML = contenido. CSS = diseño. Los dos trabajan juntos.
