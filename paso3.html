🎨 Paso 3: Dar estilo con CSS

Acá vas a aprender a ponerle colores, ordenar las cosas y hacer que se vea bien en celular.

🔗 Cómo conectar el CSS al HTML

En el <head> de tu HTML tenés que poner esta línea:

<link rel="stylesheet" href="css/style.css">

Esto le dice al HTML: "usá este archivo para los estilos".

✏️ Cómo se escribe CSS

/* Esto es un comentario en CSS */

/* SELECCIONÁS algo y le ponés estilo */
selector {
    propiedad: valor;
    propiedad: valor;
}

/* Ejemplos */
h1 {
    color: blue;        /* texto azul */
    font-size: 32px;    /* tamaño 32 píxeles */
}

.destacado {
    background: yellow; /* fondo amarillo */
}

#logo-principal {
    width: 150px;       /* ancho 150 píxeles */
}

🎯 Tipos de selectores

Los selectores son cómo "apuntás" a los elementos para cambiarles el estilo:

Tipo Ejemplo ¿Qué selecciona?
Etiqueta h1 { ... } Todos los <h1> de la página
Clase .tarjeta { ... } Todos los elementos con class="tarjeta"
ID #menu { ... } El elemento con id="menu" (único)

📱 Diseño responsivo (que se vea en celular)

Con esta "receta" mágica tu sitio se adapta solo a celulares. Ponelo al final de tu CSS:

/* Para pantallas de hasta 768px (tablets y celulares) */
@media (max-width: 768px) {

    body {
        font-size: 16px;
    }

    /* Las columnas se apilan una abajo de otra */
    .columnas {
        flex-direction: column;
    }

    /* El menú se ordena en vertical */
    nav ul {
        flex-direction: column;
        text-align: center;
    }
}
📌 Explicación: @media (max-width: 768px) significa "si la pantalla es más chica que 768 píxeles (tamaño de un celular), aplicá estos estilos".

✨ Animaciones y transiciones

Con esto tus tarjetas aparecen con un efecto suave. Primero definís la animación y después la aplicás:

1. Definir la animación (ponelo en tu CSS):

@keyframes aparecer {
    from {
        opacity: 0;
        transform: translateY(20px);
    }
    to {
        opacity: 1;
        transform: translateY(0);
    }
}

2. Aplicarla a un elemento:

.tarjeta {
    animation: aparecer 0.8s ease forwards;
}

Transiciones (efecto al pasar el mouse):

.boton {
    background: blue;
    transition: background 0.3s ease;
}

.boton:hover {
    background: darkblue;
}

:hover = cuando pasás el mouse por encima.

📐 Flexbox (para ordenar cosas)

Flexbox es la forma más fácil de ordenar elementos en filas o columnas:

.contenedor-flex {
    display: flex;          /* activa flexbox */
    gap: 20px;              /* espacio entre elementos */
    justify-content: center; /* centrar horizontalmente */
    flex-wrap: wrap;        /* si no entra, baja a la siguiente línea */
}
💡 Resumen rápido de CSS:
  • Todo va en el archivo style.css dentro de la carpeta css/
  • Usá clases (.nombre) para estilos que se repiten
  • Usá ID (#nombre) para algo único
  • El responsive se hace con @media
  • Las animaciones con @keyframes + animation
  • Los efectos hover con :hover
