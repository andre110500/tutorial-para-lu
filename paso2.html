📝 Paso 2: Escribir el HTML

Acá te explico cada parte del HTML que necesitás para tu proyecto.

📄 Estructura básica de todo HTML

Toda página HTML empieza con esta estructura (copiala siempre):

<!DOCTYPE html>
<html lang="es">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Título de tu página</title>
    <link rel="stylesheet" href="css/style.css">
</head>
<body>
    ← Acá va todo el contenido →
</body>
</html>

Explicación rápida:

  • <head>: información interna (título, estilos)
  • <body>: todo lo que se ve en la página
  • <link rel="stylesheet">: conecta el CSS

🏠 Cabecera (header) con menú de navegación

Esta parte va arriba de todo y tiene el logo + los links a las páginas:

<header>
    <div class="contenedor">
        <a href="index.html" class="logo">Mi Sitio</a>
        <nav>
            <ul>
                <li><a href="index.html">Inicio</a></li>
                <li><a href="proyectos.html">Proyectos</a></li>
                <li><a href="contacto.html">Contacto</a></li>
            </ul>
        </nav>
    </div>
</header>

📊 Tabla de precios o servicios

La tabla es obligatoria. Acá un ejemplo de tabla de precios:

<h2>Tabla de precios</h2>

<table>
    <thead>
        <tr>
            <th>Servicio</th>
            <th>Descripción</th>
            <th>Precio</th>
        </tr>
    </thead>
    <tbody>
        <tr>
            <td>Diseño de logo</td>
            <td>Logo profesional</td>
            <td>$15.000</td>
        </tr>
        <tr>
            <td>Sitio web</td>
            <td>5 páginas HTML</td>
            <td>$50.000</td>
        </tr>
    </tbody>
</table>
📌 Explicación:
  • <table> crea la tabla
  • <thead> = fila de encabezados (títulos)
  • <tbody> = filas con datos
  • <tr> = una fila
  • <th> = celda de título (negrita)
  • <td> = celda normal

📋 Formulario de contacto

El formulario también es obligatorio. Acá el ejemplo completo:

<h2>Contactame</h2>

<form action="#" method="POST">

    <label for="nombre">Nombre</label>
    <input type="text" id="nombre" name="nombre" required>

    <label for="email">Email</label>
    <input type="email" id="email" name="email" required>

    <label for="mensaje">Mensaje</label>
    <textarea id="mensaje" name="mensaje" rows="5" required></textarea>

    <button type="submit">Enviar</button>

</form>
📌 Explicación:
  • <form> crea el formulario
  • <label> = texto que explica el campo
  • <input> = campo donde se escribe
  • <textarea> = campo grande para mensajes
  • <button type="submit"> = botón de enviar
  • required = el campo es obligatorio

🦶 Pie de página (footer)

Va al final de todas las páginas:

<footer>
    <div class="contenedor">
        <p>&copy; 2026 Mi Sitio. Todos los derechos reservados.</p>
    </div>
</footer>
💡 Tips:
  • Usá <header>, <nav>, <section>, <footer> (son etiquetas semánticas, suman puntos)
  • El atributo class="..." sirve para darle estilo con CSS después
  • El atributo id="..." es como el DNI de un elemento (único)
